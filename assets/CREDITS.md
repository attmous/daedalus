# Asset credits

## Bust photograph — `source/plato-bust.jpg`

- **Subject:** Marble bust of Plato. Roman copy (1st c. CE) of a Greek original (350–340 BCE).
- **Held by:** Altes Museum, Berlin.
- **Source:** [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Marble_bust_of_the_Greek_philosopher_Plato._Roman_copy_(1st_century_CE)_of_an_original_(350-340_BCE)._Altes_Museum,_Berlin.jpg)
- **Status:** Public domain (faithful photographic reproduction of a 2-D public-domain work; see Commons file page for exact licensing notice).
- **Used in:** `assets/daedalus-banner.gif` (composited with editorial treatment).

The subject of the bust is Plato; we use it as a generic "classical wise craftsman"
stand-in for Daedalus — there is no canonical surviving bust of Daedalus himself.
